# Vue + Vite Template

A minimal Vue 3 starter with Vite 7 and TypeScript.

## Tech Stack

- **Framework**: Vue 3.5
- **Build Tool**: Vite 7
- **Language**: TypeScript 5.9
- **Type Checking**: vue-tsc
- **Package Manager**: npm
- **Build Output**: `dist`
- **Dev Port**: 5173

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy to Vercel or Netlify |

## Project Structure

```
├── src/
│   ├── App.vue        # Main App component
│   ├── main.ts        # Entry point
│   ├── style.css      # Global styles
│   └── components/    # Vue components
├── public/            # Static assets
├── index.html         # HTML template
└── vite.config.ts     # Vite configuration
```

## Quick Commands

```bash
# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build

# Preview production build
npm run preview
```

## Features

- Vue 3.5 with Composition API
- Vite 7 with fast HMR
- TypeScript with vue-tsc
- Single File Components (SFC)
- `<script setup>` syntax support
